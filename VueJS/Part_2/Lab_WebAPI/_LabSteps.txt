Lab: 以 Node.js 建構 Web API


0.  啟動 Web API 伺服器


0.1 滑鼠右鍵點按 Lab_WebAPI | Open in Integrated Terminal


0.2 在終端機視窗輸入：
    npm install


0.3 繼續在終端機視窗輸入:
    node app.js


0.4 啟動瀏覽器，連接到:
    http://localhost:3000



0.5 在瀏覽器連接 http://localhost
	  試用本系統各項增查修刪功能

    <Note>
    考量到簡化教室環境，
    目前的 Web API Sever 直接以讀寫
    檔案的方式而非存取資料庫。 

    在 課程資料夾/More/Lab_WebAPI 另有一個使用
    MySQL 資料庫的例子。
